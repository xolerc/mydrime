# Xoleric — Interactive Portfolio

An immersive, interactive portfolio website featuring a dual-layer cursor reveal effect, animated neon welcome text, orbiting social icons, and connected neural particles.

**Live Demo:** [xolerc.github.io/mydrime](https://xolerc.github.io/mydrime/)

---

## Features

- **Cursor Reveal Effect** — Move your mouse to reveal a hidden image through a smooth radial mask
- **Neon Welcome Text** — Letter-by-letter animated text with glowing cyan pulse
- **Orbiting Social Icons** — Six social media icons rotating around the viewport
- **Neural Particles** — Connected black dots with gold lines that repel from cursor and reveal circle
- **Mobile Support** — Touch-based reveal with responsive layout

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Structure | HTML5 |
| Styling | CSS3 (mask-image, backdrop-filter, animations) |
| Logic | Vanilla JavaScript (Canvas API, requestAnimationFrame) |

## Getting Started

```bash
git clone https://github.com/xolerc/mydrime.git
cd mydrime
open index.html
```

No build tools or dependencies required — pure HTML/CSS/JS.
